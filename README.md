# 📊 NSE Stock Backtester

Advanced trading strategy backtesting platform for NSE (National Stock Exchange) stocks.

## Features

✅ **Nifty 50 Stocks** - All major Indian stocks  
✅ **Multiple Strategies** - RSI, Moving Average, Bollinger Bands, VWAP  
✅ **Technical Indicators** - SMA20, SMA50, SMA200, RSI, VWAP, Bollinger Bands, MACD  
✅ **Interactive Charts** - Real-time price visualization  
✅ **Trading Signals** - Detailed buy/sell signals with analysis  
✅ **Backtest Metrics** - Return %, trades count, price analysis

## Quick Start

### On Replit (Recommended)

1. Click "Import from GitHub"
2. Paste: `https://github.com/tempram75-lab/nse-backtester`
3. Click "Import"
4. Click "Run" (green button)
5. Open the URL in your browser
6. Start backtesting! 🚀

### Locally

```bash
git clone https://github.com/tempram75-lab/nse-backtester.git
cd nse-backtester
pip install -r requirements.txt
python app.py
